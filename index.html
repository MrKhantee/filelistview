    Android File ListView

    This is a simple file browser meant to be integrated into Android apps.

    Integrating this project

    Integrating this project requires three steps:

    • Put the contents of the /src folder somewhere in your /src folder (and, if necessary, change the paths).
    • Take the image files from the /res folder, and put them in your own res folder
    • Take the strings from the values.xml file and add them to the end of your own values.xml file
    • Update the imports in this project as necessary (you will need to update the R import to fit your project).
